# PoseGuard
AI 기반 실시간 자세 교정 및 거북목 방지 시스템

## 프로젝트 소개
PoseGuard는 노트북 웹캠을 활용하여 사용자의 상체 자세를 실시간으로 분석하고, 잘못된 자세를 감지하여 즉각적인 피드백과 맞춤형 스트레칭 가이드를 제공하는 AI 기반 헬스케어 프로그램입니다.

장시간 학습 환경에서 발생하는 거북목 및 자세 불균형을 예방하고, 데이터 기반 리포트를 통해 올바른 자세 습관 형성을 돕는 것을 목표로 합니다.


## 주요 기능

### 실시간 자세 분석
- OpenCV + MediaPipe 기반 관절 좌표 추출
- 목, 어깨 위치를 기반으로 자세 상태 분석

### 자세 이상 감지 및 알림
- 거북목 및 라운드 숄더 감지
- 일정 시간(5초) 이상 지속 시 경고 메시지 출력

### 스트레칭 가이드 제공
- 동일 부위 반복 이상 감지 시 스트레칭 안내
- 텍스트 및 이미지 기반 가이드 제공

### 결과 리포트 및 시각화
- 자세 유지율 및 알림 횟수 제공
- Matplotlib 기반 그래프 시각화

### AI 건강 코멘트
- Google Gemini API 활용
- 사용자 데이터 기반 맞춤형 피드백 생성


## 시스템 구조
- OpenCV (영상 처리)
→ MediaPipe (좌표 추출)
→ 자세 분석 로직
→ 알림 및 피드백 출력
→ Firebase 저장
→ 결과 시각화 + AI 코멘트

- 5~7주차: 자세 인식 기능 구현
